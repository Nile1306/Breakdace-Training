# mytodolist-backend
Breakdance Footwork Training App
Beschreibung
Diese Webanwendung unterstützt das Training von Breakdance-Footwork sowie einfache Fitness-Workouts.
Die App bietet einen Footwork-Drill-Modus mit Timer und eine ToDo-Liste für tägliche Übungen wie Push-ups oder Squats.

Funktionen
 Footwork-Drill 
 Timer mit 10s Countdown, 40s Training und 20s Pause (3 Runden)
 Einfache Sprachansagen (Start, Rest, Finish)
 Hintergrundmusik während des Trainings
 ToDo-Liste für Workouts (hinzufügen und abhaken)
